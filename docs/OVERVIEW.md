# Архитектура системы

## Цель

Автономная утренняя рутина без взрослого промптирования. Система реплицирует функцию взрослого — напоминает, ждёт подтверждения, реагирует на просрочку — и постепенно передаёт ответственность ребёнку.

## Контекст

- Мальчик 13–14 лет, атипичное нейроразвитие
- Нарушена инициация действий и переходы между задачами
- Хорошо реагирует на короткие задачи по одной
- Читает эмоциональное состояние взрослых
- Мотиваторы: велосипедные прогулки, мультфильм Фиксики
- Взрослый (Сергей) ведёт утреннюю рутину 4 года

## Принципы

1. **Естественные последствия** — голод, отсутствие компьютерного времени — эффективнее технического принуждения
2. **Одна задача за раз** — никаких списков, только текущий шаг
3. **Система учит самостоятельности**, а не создаёт постоянные костыли
4. **Якорный звук** — мягкий двойной колокол перед каждым голосовым промптом формирует условный рефлекс внимания

## Фазы дня

```
До 8:00        Утро (morning)   — тёплый жёлтый свет, шаги рутины
8:00–17:00     Школа (school)   — нейтральный белый, таймер до возвращения  
17:00–21:00    Дома (home)      — обратный отсчёт до 22:00
21:00–22:00    Вечер            — пурпурный → жёлтый → красный по мере приближения сна
```

## Утренние шаги

1. Умылся
2. Оделся
3. Рюкзак (с едой)
4. Завтрак
5. Обулся

## Механика оболочки

- Кнопка "Готово" заблокирована первые **60 секунд** на каждом шаге
- Голосовой промпт повторяется каждые **2 минуты** если шаг не подтверждён
- Первичный голос: MP3-записи мамы
- Fallback: Web Speech API, русский женский голос
- Якорный звук предшествует каждому промпту
