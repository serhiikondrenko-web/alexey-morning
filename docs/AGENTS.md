# Агентные компоненты

Оболочка состоит из нескольких автономных логических агентов, каждый с чёткой зоной ответственности.

---

## 1. PhaseAgent — агент фазы дня

**Зона:** определяет текущую фазу (morning / school / home / evening)  
**Триггер:** каждую минуту проверяет системное время  
**Выход:** переключает UI и передаёт команды LightAgent

```
До 8:00      → phase: morning
8:00–17:00   → phase: school
17:00–21:00  → phase: home
21:00–22:00  → phase: evening
```

**Логика вечерней фазы:**
- 21:00–21:30 → фиолетовый свет, спокойный тон
- 21:30–21:45 → жёлтый, мягкое предупреждение
- 21:45–22:00 → красный, чёткое напоминание

---

## 2. StepAgent — агент шагов рутины

**Зона:** управляет последовательностью утренних шагов  
**Активен:** только в фазе morning

**Поведение:**
- Показывает один текущий шаг
- Блокирует кнопку "Готово" первые 60 секунд
- При нажатии "Готово" — переходит к следующему шагу
- После последнего шага — завершает утреннюю фазу

**Защита от читерства:**
- Кнопка недоступна первые 60 сек (исключает быстрое тапание)
- Таймер не перематывается (ранее был баг с видео-таймером)

---

## 3. VoiceAgent — голосовой агент

**Зона:** голосовые напоминания  
**Триггер:** каждые 2 минуты если текущий шаг не подтверждён

**Цепочка воспроизведения:**
1. Якорный звук (мягкий двойной колокол)
2. MP3-файл с голосом мамы (если загружен)
3. Fallback: Web Speech API, русский женский голос

**Файлы голоса мамы (ожидают записи):**
```
voice/umylsya.mp3
voice/odelsa.mp3
voice/ryukzak.mp3
voice/zavtrak.mp3
voice/obulsya.mp3
voice/anchor.mp3   ← якорный звук
```

## 4. LightAgent — агент освещения

**Зона:** управление лампой Yeelight W3 через LAN API  
**Протокол:** TCP JSON на порту 55443  
**IP лампы:** `192.168.1.209` (рекомендуется зафиксировать в роутере)

**Сценарии освещения:**

| Сценарий | Температура | Яркость | Когда |
|---|---|---|---|
| Пробуждение | 2700K тёплый | 30% | Начало morning фазы |
| Активный шаг | 4500K нейтральный | 80% | Каждый шаг рутины |
| Просрочка | 6500K холодный | 100% + мигание | Шаг не выполнен > N мин |
| Школа | 4500K | 70% | Фаза school |
| Вечер спокойный | 2700K | 40% | 21:00 |
| Вечер тревога | 6500K | 100% | 21:45+ |

**Команды LAN API:**
```javascript
// Включить
{"id":1,"method":"set_power","params":["on","smooth",500]}

// Температура цвета
{"id":2,"method":"set_ct_abx","params":[2700,"smooth",1000]}

// Яркость
{"id":3,"method":"set_bright","params":[80,"smooth",800]}

// Мигание (color flow)
{"id":4,"method":"start_cf","params":[0,0,"500,2,6500,100,500,2,6500,10"]}

// Остановить мигание
{"id":5,"method":"stop_cf","params":[]}
```

**Важно:** в приложении Yeelight должен быть включён **LAN Control**.

## 5. ScheduleAgent — агент расписания

**Зона:** чтение конфигурации из Google Sheets  
**Интервал:** polling каждые 2–3 минуты  
**Spreadsheet ID:** `1cqW87tHE6TQ0piTmaO6wNjJ-deze8bxA5ZbM3YOPKps`

**Листы:**

| Лист | Содержимое |
|---|---|
| `schedule` | Время фаз, дни недели |
| `overrides` | Разовые переопределения (каникулы, болезнь) |
| `steps` | Список шагов и их порядок |
| `voice` | Тексты для TTS fallback |

**Статус:** Apps Script (`setup_shell_config.gs`) создан, подключение HTML к Sheets — в процессе.

## 6. GamificationAgent — агент геймификации

**Зона:** награды и мотивация  
**Символ:** иконки ⚙️ (Фиксики)

**Механика:**
- Стрик — счётчик подряд выполненных дней
- Прогресс-бар недели (5 школьных дней)
- Награда за 5/5 дней → велосипедная прогулка
- Хранение: localStorage планшета
